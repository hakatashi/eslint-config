# @hakatashi/eslint-config



## Install

    npm install @hakatashi/eslint-config --save-dev

or

    yarn add @hakatashi/eslint-config --dev

## Usage

Your `.eslintrc`:

```json
{
	"extends": "@hakatashi"
}
```

Or for typescript:

```json
{
	"extends": "@hakatashi/eslint-config/typescript"
}
```
