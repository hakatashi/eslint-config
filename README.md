# @hakatashi/eslint-config

[![Greenkeeper badge](https://badges.greenkeeper.io/hakatashi/eslint-config.svg)](https://greenkeeper.io/)

## Install

    npm install @hakatashi/eslint-config --save-dev

or

    yarn add @hakatashi/eslint-config --dev

## Usage

Your `.eslintrc`:

```json
{
	"extends": "@hakatashi"
}
```
